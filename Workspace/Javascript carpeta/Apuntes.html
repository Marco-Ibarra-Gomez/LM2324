Tipos de datos: se dividen en 3 Tipos - : Son los números, decimales o enteros. - : Son valores de texto (hacemos referencia a un texto) y debe ir siempre entre comillas dobles o simples ("" ''). - : Tienen que ser True o False (no hay más posibilidades). Los tipos de datos están relacionados con las variables. Una es "un espacio en la memoria del ordenador (RAM) donde se almacenará un valor que podrá cambiar durante la ejecución del programa". Para declarar una variable hay que usar "var" y a continuación el nombre de la variable, por ejemplo: "var puntuación;". Para dar hay que seguir las siguientes reglas: - La variable comienza por letra, símbolo $ o _ (puntuacion, $puntuacion, _puntuacion) - Contiene solo letras, números, $ y _ (puntuacion1, puntuacion_1, ...) - Son case sensitive (distingue entre mayúsculas y minúsculas) (por ejemplo, Puntuacion y puntuacion no son iguales) - No deben ser palabras reservadas (estas son palabras que forman parte del lenguaje, por ejemplo: alert, var, ... y no se pueden declarar de la forma "var alert;") - Son descriptivas Las variables se pueden declarar en distintas lineas o en una misma linea como se ven en los siguientes 4 ejemplos: O también se puede hacer esto: Tipos de en Javascript: - Básicos: (+): realiza una operación de suma (también se usa para concatenar/unir dos valores, un string y un valor numérico) (-): realiza una operación de resta (*): realiza una operación de multiplicación (/): realiza una división Existen : - De entrada de datos (prompt): permite introducir datos desde fuera del programa a dentro del programa y almacenarse en una variable. - De salida de datos(alert): Con este, la información de una variable es capaz de salir del programa hacia fuera en forma de ventana para que la vea el usuario. : (+=): Incrementa en X el valor de una variable (-=): Decrementa en X el valor de una variable (*=): Multiplica en X el valor de una variable (/=): Divide en X el valor de una variable (++): Incrementa en 1 el valor de una variable (--): Decrementa en 1 el valor de una variable Los se usan para reservar varios espacios en la RAM del PC donde iremos almacenando sucesivamente valores, además, esos espacios están unidos o empaquetados en un array. Un Array es una gran variable donde se almacenan varios valores. En vez de permitir guardar en una variable un valor, permite guardar un grupo de valores (es como un archivador). Por ejemplo, si tengo un array de 4 elementos, el 4to elemento es el [3] (siempre es uno menos). Para acceder a lo que hay en un índice del Array hay que escribir el nombre del array y luego la posición. Las son herramientas para manipular un Array: - Length: Nos dice la longitud o número de elementos que tiene un array (articulos.length = 4;) - Push: Nos permite agregar elementos al array al final - Unshift: Nos permite agregar elementos al array al principio - Pop: Nos permite eliminar elementos al final del array - Shift: Nos permite eliminar elementos al principio del array : Todos los objetos tienen características (ancho, alto, calor, ...) y capacidades (sostener objetos, lavar ropa, arrancar, ...). Como tal, no son característicias y capacidades, sino propiedades y métodos, respectivamente: - Propiedades: Cómo es el objeto (ejemplo: style.width, boton.focus, window.alert, ...) - Métodos: Qué puede hacer el objeto (ejemplo: getElementById, ...)
